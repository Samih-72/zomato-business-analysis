# Zomato Power BI Dashboard

## Project Overview

This project analyzes Zomato restaurant data using Power BI.

## KPIs

- Total Restaurants
- Average Rating
- Average Cost for Two
- Online Delivery Availability
- Restaurant Distribution by City

## Tools Used

- Power BI
- Excel / CSV
- Data Cleaning
- Data Visualization

## Key Insights

- Identified top-performing cities.
- Analyzed rating distribution.
- Compared online delivery adoption.
- Evaluated restaurant pricing patterns.
